This project aims to build an intelligent system that classifies emails as Spam or Ham (Not Spam) using machine learning techniques. It leverages Natural Language Processing (NLP) to analyze and understand the textual content of emails.

The dataset is preprocessed by handling missing values and converting categorical labels into numerical form. The core feature extraction technique used is TF-IDF (Term Frequency–Inverse Document Frequency), which transforms raw text into meaningful numerical vectors by assigning importance to relevant words.

Multiple machine learning models such as Naive Bayes, Logistic Regression, and Support Vector Machine (SVM) are trained and evaluated to identify the best-performing model. Among these, Naive Bayes performs particularly well due to its effectiveness in text classification tasks.

The model is evaluated using key performance metrics including accuracy, precision, recall, and F1-score, ensuring reliable and balanced predictions.
